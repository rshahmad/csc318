My project in developing system and web.

During my internship at Turbine Technique, I was placed in System Development Department(SDD). SDD need to make a system that can record any fault report regarding to the system and items when parliament staff make a report. The parliament staff can only report the system and items that only listed in maintenance contract. The system can store all the report made by parliament staff and SDD team can view the report from the system. There are two type of users for this system which is parliament staff(user) and SDD staff(admin).

Homepage for Fault Reporting System for Dewan Parlimen Sementara(DPS)

Click 'User' or 'Admin' below to view the interface of this system.


Interface for User

User can fill the form to register in order to use the system

User can reset password if they forgot their password

User can enter email and password to login the system

Interface for user after login

User can update their details

User can change their password

User can fill the form to make a report

User can view their own report

User can view other users' report

User can search other users' report

Interface for Admin

Admin can enter username and password to login the system

Interface for admin after login

Admin can view list of users

Admin can view the record of users

Admin can register password for users

Admin can delete users' account

Admin can view list of report made by users

Admin can view the record of report

Admin can update the status of report

Admin can search the report made by users
